## The Equation

```
S = min(Stock Margin, FX Margin) ≥ 0
```

A civilization zone can sustain itself when and only if **both** margins stay non-negative. The weakest margin determines survival.

---

## Components

### Stock Margin (Resource Balance)

```
Stock Margin = I + P + R − (C + Eg)
```

| Variable | Meaning |
|----------|---------|
| I | Import (daily) |
| P | Production gain (daily) |
| R | Recycling (daily, FREE) |
| C | Consumption (daily) |
| Eg | Export Goods (daily) |

### FX Margin (External Capacity)

```
FX Margin = Es + Eg − I
```

| Variable | Meaning |
|----------|---------|
| Es | Export Service (daily, FC) |
| Eg | Export Goods (daily) |
| I | Import (daily) |

## Why Two Margins?

THOOTB zones face a **dual constraint**:

1. **Physical resources** must not deplete (Stock Margin)
2. **Foreign currency** must not run out (FX Margin)

Either constraint failing causes collapse. The system is only as strong as its weakest margin.

> "A civilization is only as strong as its weakest margin." — Copilot
> "Export buys time. Recycling buys eternity." — Grok
